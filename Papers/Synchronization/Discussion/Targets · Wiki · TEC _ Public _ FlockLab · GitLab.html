FlockLab Targets

Here is a list of currently mounted FlockLab targets (aka Sensor Nodes aka Devices under test):

Node Name Manufacturer MCU Radio #Targets Installed (indoor / outdoor) Name in XML Comments More Info
TinyNode184 Shockfish SA MSP430F2417 (16 MHz) Semtech SX1211 (868/915 MHz) 14 tinynode 512kB external flash; On-board chip antenna (footprint for SMA/MMBX connector) Link
Tmote Sky (TelosB) Moteiv MSP430F1611 (8 MHz) CC2420 (IEEE 802.15.4, 2.4 GHz) 27 tmote On-board chip antenna (footprint for SMA connector); On-board humidity, temperature and light sensors Link
OpenMote OpenMote ARM 32-bit Cortex-M3 (32 MHz) (CC2538 SoC) CC2520-like (2.4 GHz) (CC2538 SoC) 9 openmote Runs up to 32 MHz and includes 32k RAM and 512k flash memory; compatible with IEEE 802.15.4-2006 Link
DPP (Dual Processor Platform) custom CC430F5147, MSP432P401R (Rev.B), MSP430FR5969 (BOLT) CC430 SoC, CC1101-based (868 MHz) 29 dpp All 3 cores can be programmed. Link
DPP2 LoRa custom STM32L433CC SX1262 (868 MHz) 28 / 2 dpp2lora Link

Other sensors can be attached to the testbed easily due to FlockLabs generic hardware interface. Contact us for more information.
